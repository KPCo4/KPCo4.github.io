Lab Setup Report

Documentation of a controlled malware analysis environment and its configuration.

Date: 14-08-2025

Contributors: Kye P.C.

Lab Purpose: Safe environment for malware detonation and behavioural analysis.

Virtualization: VirtualBox 7.1 running on Ubuntu 22.04 (host)

Setup Process

This section outlines the steps I've taken to configure the lab environment and ensure isolation from external networks.

  1. Host Preparation: Installed Ubuntu 22.04 LTS with all security updates applied.
  2. Virtualization Platform & Topology: Configured VirtualBox on the Ubuntu host to run two isolated VMs:
    • Windows 10 (analysis VM) - primary analysis target (8 GB RAM, 4 cores)
    • Ubuntu (INetSim server) - provides simulated internet services (DNS, HTTP, SMTP) for the analysis VM.
    Both VMs are connected via a VirtualBox **host-only** network so the analysis VM uses INetSim for DNS and service responses while remaining isolated from the real LAN and internet.
  3. Guest Creation: Deployed a Windows 10 (x64) VM with 8 GB RAM, 4 cores, and no shared clipboard or drag-and-drop.
  4. Analysis Tools Installation:
    • CFF Explorer
    • CyberChef v10
    • Ghidra
    • IDA Free
    • Regshot
    • Resource Hacker
    • Sysinternals Suite
    • Wireshark 4.4
    • x64dbg
  5. Snapshot Baseline: Created a clean “Tools Installed" snapshot before loading any samples.

Validation & Testing

After initial setup, validation was performed to confirm the lab's isolation, functionality, and stability.

  • Network Isolation: Verified guest VM network isolation using host-only adapter. Internet access blocked; internal communication confirmed.
  • Tool Functionality: All analysis tools launched successfully and captured expected data. No cross-tool interference observed.
  • Snapshot Rollback: Snapshot rollback verified. Environment returns to known-clean state.
  • Performance: Performance stable at 4 vCPUs, 8GB RAM.

No anomalies observed during validation. Environment is considered safe for dynamic malware analysis.
